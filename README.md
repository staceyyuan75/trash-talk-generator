# 幹話產生器

心累了嗎？來選個對象說出內心話吧。這個專案使用 Node.js + Express，後端函式依照使用者選的幹話對象，隨機產生配對語句。

## 開始 Getting Started

本專案主要運行在電腦本機端，在開始運行專案前請先依照步驟安裝相關軟體與套件，即可瀏覽專案結果。

### 環境建置需求/相關套件 Prerequisites

- **[Node.js](https://nodejs.org/)**
- **[npm](https://www.npmjs.com/)** (安裝好 Node.js 後就包含在內)
- **[Express](https://www.npmjs.com/package/express)**
- **[Express Handlebars](https://www.npmjs.com/package/express-handlebars)** 專案模版工具
- **[body-parser](https://www.npmjs.com/package/body-parser)** 取得表單 POST 資料
- **[nodemon](https://nodemon.io/)**

### 安裝流程 Installation

1. 開啟終端機 (Terminal)，在電腦本機端使用 git clone 此專案

```
git clone https://github.com/staceyyuan75/trash-talk-generator.git
```

2. 在終端機 (Terminal) 輸入指令，切換到下載好的專案資料夾

```
cd trash-talk-generator
```

3. 安裝 npm 套件

```
npm install
```

4. 執行指令，啟動伺服器

```
npm run dev
```

5. 當終端機 (Terminal) 顯示 `Express is listening on localhost:3000`，表示啟動成功！

啟動成功後，可以開啟瀏覽器，輸入網址 **[http://localhost:3000](http://localhost:3000)** 即可開始使用幹話產生器。
